# 3D Printing Designs

A collection of free 3D printable CAD designs created with OpenSCAD and shared as parametric models.

## Overview

This repository contains open-source 3D printing designs that can be customized and printed for personal or commercial use. All models are designed using [OpenSCAD](https://openscad.org/), a programmer's solid 3D CAD modeller, which allows for fully parametric and customizable designs.

## Features

- **Parametric Designs**: All models are fully parametric, allowing you to customize dimensions and features
- **OpenSCAD Source**: Editable `.scad` files for complete control over design modifications
- **Free and Open**: All designs are freely available under the MIT License
- **MakerWorld Integration**: Models are published on MakerWorld for easy access and community sharing

## Getting Started

### Prerequisites

To work with these designs, you'll need:
- [OpenSCAD](https://openscad.org/downloads.html) - Download and install the latest version
- A 3D slicer software (e.g., PrusaSlicer, Cura, or Bambu Studio)
- A 3D printer or access to 3D printing services

### Using the Designs

1. **Clone the repository**:
   ```bash
   git clone https://github.com/NF-Software-Inc/3D-Printing-Designs.git
   cd 3D-Printing-Designs
   ```

2. **Open a design in OpenSCAD**:
   - Launch OpenSCAD
   - Open the desired `.scad` file
   - Modify parameters as needed
   - Render and export to STL format

3. **Print the model**:
   - Import the STL file into your slicer software
   - Configure print settings for your printer
   - Slice and print

## Contributing

We welcome contributions! If you have ideas for improvements or new designs:

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-design`)
3. Commit your changes (`git commit -m 'Add amazing design'`)
4. Push to the branch (`git push origin feature/amazing-design`)
5. Open a Pull Request

## Design Guidelines

When contributing designs:
- Use descriptive variable names in OpenSCAD code
- Include comments explaining key parameters
- Provide reasonable default values
- Test that the model renders and exports correctly
- Include any special printing requirements or recommendations
